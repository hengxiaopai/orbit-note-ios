# Orbit Note

Orbit Note is a SwiftUI iOS app for spatial life orbit journaling.

It helps users record the people, projects, events, and emotions that orbit around their daily life, using distance, energy, and intensity to visualize personal attention and life balance.

## Status

Current version: `v0.2.1`

- SwiftUI MVP completed
- SwiftData local persistence completed
- Xcode build pre-fixes completed
- Waiting for real macOS / Xcode / Simulator verification

## Design Direction

**Radium Noir + Aurora Glass**

- OLED near-black background
- Thin orbit lines
- Cyan / pink-orange / mist-purple energy colors
- Glass cards
- Calm motion
- Native iPhone-first interaction

## Current Features

- 4 main tabs: Orbit, Add, Timeline, Me
- Today orbit visualization
- Add orbit entries
- Edit orbit entries
- Delete orbit entries
- Recent 7-day timeline
- SwiftData local persistence
- Clear local data
- Restore sample data

## Tech Stack

- SwiftUI
- SwiftData
- iOS 17+
- Local-first data model
- No backend
- No login
